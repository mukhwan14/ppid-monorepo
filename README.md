# PPID Monorepo
Aplikasi Pengelolaan dan Pelayanan Informasi Publik PPID Garut

## Struktur
- **ppid-frontend** → Aplikasi frontend (Next.js)
- **ppid-backend** → Aplikasi backend (Node.js/Express)

## Cara Menjalankan
### Frontend
cd ppid-frontend
npm install
npm run dev

### Backend
cd ppid-backend
npm install
npm run dev
